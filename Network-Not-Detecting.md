# Network Not Detecting

## Problem
Computer cannot detect network or shows “No Internet Access”.

## Possible Causes
- Faulty Ethernet cable
- Disabled network adapter
- Incorrect IP configuration
- DNS issue
- Router or switch failure

## Diagnosis Steps
1. Run ipconfig /all
2. Check adapter status in Network Settings
3. Test with another cable or port
4. Ping gateway address
5. Flush DNS cache

## Resolution
- Enable network adapter
- Renew IP address
- Replace cable or switch port
- Reset TCP/IP stack
- Restart network devices

## Prevention
- Maintain proper cable management
- Use DHCP monitoring
- Document network topology

## Business Impact
Loss of connectivity stops access to shared systems, email and cloud services.
