# Printer Not Connecting

## Problem
User cannot print to the office network printer. Printer appears offline or not listed.

## Possible Causes
- Printer disconnected from network
- Incorrect IP configuration
- Print spooler service stopped
- Driver corruption
- Permission issues

## Diagnosis Steps
1. Confirm printer is powered on
2. Check network cable or WiFi connection
3. Ping the printer IP address
4. Restart Print Spooler service
5. Verify printer driver installation
6. Check if user has access permissions

## Resolution
- Restart printer and workstation
- Reinstall printer driver
- Re-add printer via IP address
- Restart Print Spooler service
- Confirm user is in correct access group

## Prevention
- Maintain updated printer drivers
- Assign printers via Group Policy
- Monitor network printer status regularly

## Business Impact
Printing downtime affects documentation, invoicing and reporting workflows, causing operational delays.
