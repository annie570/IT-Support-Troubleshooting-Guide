# PC Slow Performance

## Problem
User reports computer takes long to start, open programs, or respond.

## Possible Causes
- High startup programs
- Low disk space
- Malware or unwanted software
- Insufficient RAM
- Pending Windows updates

## Diagnosis Steps
1. Check Task Manager for CPU/RAM usage
2. Review startup applications
3. Check disk storage level
4. Run antivirus scan
5. Check Windows update status

## Resolution
- Disable unnecessary startup apps
- Remove temporary files
- Uninstall unused programs
- Install pending updates
- Upgrade RAM if required

## Prevention
- Enforce endpoint maintenance policy
- Schedule regular cleanup scripts
- Educate users on software installation practices

## Business Impact
Slow machines reduce employee productivity and increase support tickets.
