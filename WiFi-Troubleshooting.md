# WiFi Connectivity Issues

## Problem
User cannot connect to corporate WiFi or connection drops frequently.

## Possible Causes
- Weak signal
- Incorrect credentials
- IP conflict
- Driver issue
- Access point overload

## Diagnosis Steps
1. Confirm correct WiFi network
2. Forget and reconnect network
3. Run ipconfig /release and /renew
4. Update WiFi driver
5. Test connection near access point

## Resolution
- Reset network settings
- Update wireless driver
- Reassign IP
- Escalate to network team if AP overloaded

## Prevention
- Maintain access point coverage plan
- Monitor network load
- Update drivers regularly

## Business Impact
Unstable connectivity disrupts work, meetings and access to systems.
